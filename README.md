# og-image-server

## Routes

- [GET] /static : 업로드 된 이미지 스태틱 리소스 링크
- [GET] /form : 이미지 업로드 폼
- [GET] /result : 이미지 업로드 결과
- [GET] /og/:id : Opengraph 메타데이터 포함된 페이지


## Getting Started

```bash
npm -g install yarn
git clone https://github.com/8eatles/og-image-server.git
cd og-image-server
yarn && yarn start
```
